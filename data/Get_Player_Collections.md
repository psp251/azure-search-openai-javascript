# Get Player Collections

**Group:** Player/Others

**Description:** Fetches collections owned by the player.

**Route:** /v2/client/player/get-collections

**Method:** POST

## Parameters

- **userId**: Unique identifier for the user.

## Required Headers

- **api-key**: The environment-specific API key
- **Authorization**: A valid Authorization ('Bearer <token>') is required to grant access for the player's session.

## Tags

player, collections, assets, unlocks, progress, achievements, gallery, cosmetics
